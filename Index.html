User Manual: RudGameOfLife Explorer V4.8.2

1. Interface Basics

  • Res (Resolution): Sets cell size. Use smaller values for massive patterns like the Breeder.
  • Dens (Density): Percentage of cells alive at start when clicking 'New Random'.
  • Inc (Increment): Delay between generations. 0.001s is the absolute speed limit.
  • -1 / +1: Step backward (100 step history) or forward manually.

2. Build Mode

Toggle Edit Mode via the button or ESC. The grid highlights in magenta.

  • Select a pattern (e.g., Gosper Gun) and click the grid.
  • Keyboard: Use Arrows to move and Enter to place.

3. Smart Save/Load

Saves are exported as .json. The file includes the current resolution so the explorer can restore the view perfectly upon loading.
