# RudGameOfLife Explorer (V4.8.2)

A sophisticated, interactive web-based implementation of Conway's Game of Life. 
This project was designed to be both a visual simulation and a laboratory for cellular automata.

## ✨ Key Features
- **Interactive Editor:** Press `ESC` to enter Edit Mode. Draw custom patterns or place single cells.
- **Predefined Library:** Quick access to iconic patterns like the *Gosper Glider Gun*, *Pulsar*, and *Breeder*.
- **Time Manipulation:** Advance frame-by-frame, pause, or rewind up to 100 generations.
- **Smart Save/Load:** Export your universe as a `.json` file. The explorer automatically restores the correct resolution and generation count upon loading.
- **Debug Mode:** Visualize neighbor counts for each cell to understand the underlying logic.

## 🛠️ Built With
- HTML5 Canvas API
- Vanilla JavaScript
- CSS3 (optimized for dark mode and blog embedding)

## 📖 How to Use
1. **Explore:** Click 'New Random' to start a chaotic soup.
2. **Build:** Toggle 'Edit' to design your own machines.
3. **Analyze:** Use 'Debug' and slow down the 'Inc' (speed) to see the rules in action.
